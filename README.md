# Insurance-Analytics
Developed an interactive Insurance Analytics Dashboard using Power BI, SQL, and Excel to analyze policy distribution, customer demographics, premium performance, claims, payment status, and key business KPIs for data-driven decision-making.

## How I Built the Insurance Analytics Dashboard
This project was developed to analyze insurance policy data and provide actionable business insights using SQL, Excel, and Power BI.

### Project Workflow
1. Collected and reviewed the insurance dataset provided during the internship.
2. Cleaned and validated the data by handling missing values, removing inconsistencies, and ensuring accurate data types.
3. Imported the data into SQL and wrote queries to analyze policy details, customer information, claims, premiums, and payment status.
4. Loaded the processed data into Power BI and created relationships between the tables using a common key.
5. Built calculated columns and DAX measures to generate KPIs such as:
   * Total Policies
   * Total Customers
   * Total Claim Amount
   * Premium Growth Rate
   * Policies Expiring This Year
6. Designed interactive visualizations including bar charts, pie charts, line charts, KPI cards, and slicers for easy data exploration.
7. Created a user-friendly dashboard with filters for policy type, claim status, payment status, gender, and age group to enable dynamic business analysis.
8. Validated the dashboard results against the source data to ensure accuracy and consistency.

The final dashboard provides a comprehensive view of insurance operations, helping users monitor policy performance, customer demographics, claims, premium trends, and payment insights through interactive visualizations.
